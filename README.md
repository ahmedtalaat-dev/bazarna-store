# 🛒 Bazarna - Modern E-Commerce Platform

## 🚀 Overview

**Bazarna** is a modern and fully responsive e-commerce web application built using cutting-edge front-end technologies.
The project focuses on delivering a smooth, fast, and engaging shopping experience with clean UI and interactive animations.

---

## ✨ Features

* 🛍️ Dynamic product listing
* 🔍 Advanced filtering & search
* ⚡ Fast performance with Next.js
* 🎨 Clean and modern UI design
* 📱 Fully responsive across all devices
* 🎬 Smooth animations using Framer Motion
* 🧩 Reusable and scalable components
* 🛒 Products Card

---

## 🛠️ Tech Stack

* ⚛️ Next.js
* 🟦 TypeScript
* 🎨 Tailwind CSS
* 🎞️ Framer Motion
* 🎯 Lucide React Icons

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/bazarna-store.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```
